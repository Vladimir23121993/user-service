# User Service (Hibernate + PostgreSQL)

Консольное приложение на чистой Java (без Spring), которое умеет сохранять пользователей в базу данных и выполнять все основные операции: 
создание, поиск, обновление и удаление. Работает через консольное меню, для связи с базой используется Hibernate, данные хранятся в PostgreSQL.

Проект разбит на слои:
- entity — описание пользователя (id, имя, email, возраст, дата создания)
- dao — прямые операции с базой (сохранить, найти, обновить, удалить)
- service — бизнес-логика и проверки (например, что email содержит @)
- console — консольное меню для взаимодействия с пользователем
- util — настройка Hibernate (подключение к базе)

Возможности программы:
- Создавать нового пользователя
- Находить пользователя по ID
- Показывать всех пользователей
- Обновлять данные пользователя
- Удалять пользователя
- Искать по email
- Искать по имени (даже по части)

Тесты:
- 14 тестов, покрывают все методы сервиса
- Используют Mockito для изоляции от реальной БД
- Проверяют валидацию, успешные сценарии и обработку ошибок
